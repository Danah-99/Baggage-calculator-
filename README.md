# Baggage Extra Charge Calculator

## Description
This is a Java console application that calculates extra baggage charges for airline passengers.
The calculation depends on the passenger class and the bag weight.

## Features
- Supports multiple passenger groups
- Uses loops to process passengers
- Uses switch-case to handle passenger classes
- Calculates extra charges based on baggage weight limits

## Passenger Classes & Weight Limits
- *First Class (F)*: 30 kg
- *Business Class (B)*: 25 kg
- *Economy Class (E)*: 20 kg

Any extra weight is charged at *10 units per kg*.

## Technologies Used
- Java
- Scanner for user input
- Control structures (loops, if-statements, switch-case)
